GtkScintilla by codebrainz

A GTK+ based source code editing Widget.

GtkScintilla is a GTK+ widget which inherits from the excellent Scintilla GTK+ widget. The API of the Scintilla GTK+ widget is quite unusual (due to its roots as a Windows control), so GtkScintilla brings the usual GTK+ widget to the Scintilla GTK+ widget. For more information about Scintilla, see the Scintilla website.

Some of the current and future features (beyond those of Scintilla itself) include:

  • Wrapper functions for all of the Scintilla messages.
  • Helper functions for common things such as enabling line numbers and code folding.
  • Proper GObject properties to help with introspection for things like Glade and GObject-Introspection.
  • Individual signals for each of the Scintilla notifications.
  • GTK+ style properties to support controlling the styles from GTK+ resource files. (future)
  • Glade 3 support.
  • Language bindings for Python, Vala and others. (future)
  • A shared library and pkg-config file for easily linking into applications.
  • GTK-DOC documentation for the entire API. (future)

Dependencies

gtk+-2.0

Install

$ ./autogen.sh
$ ./configure
$ make
$ make install

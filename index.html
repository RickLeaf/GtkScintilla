GtkScintilla by codebrainz

A GTK+ based source code editing Widget.

GtkScintilla is a GTK+ widget which inherits from the excellent Scintilla GTK+ widget. The API of the Scintilla GTK+ widget is quite unusual (due to its roots as a Windows control), so GtkScintilla brings the usual GTK+ widget to the Scintilla GTK+ widget.

Dependencies

gtk+-2.0

Install

$ ./autogen.sh
$ ./configure
$ make
$ make install

The "char-added" signal

                                                        gint          chr,
                                                        gpointer      user_data)      : Run First

Sent when the user types an ordinary text character (as opposed to a command character) that is entered into the text. The user can decide to use this to decide to display a calltip or an auto completion list. This signal is emitted before the character has been styled so processing that depends on styling should instead be performed in the "update-ui" signal handler.
